# FFmpeg Binary

DeadAir uses FFmpeg for audio silence detection.

## Option 1: System Install (Recommended)

**Windows:**
```
winget install ffmpeg
```

**macOS:**
```
brew install ffmpeg
```

**Linux:**
```
sudo apt install ffmpeg
```

## Option 2: Portable

Download a static FFmpeg build and place the executable in this folder:

- **Windows:** Place `ffmpeg.exe` here
- **macOS/Linux:** Place `ffmpeg` here

Download from: https://ffmpeg.org/download.html
